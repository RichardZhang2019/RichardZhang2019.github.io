install-hexo

用管理员身份运行npm，安装hexo-cli

运行 Task Manager > Run new task > cmd, 选择 用管理员权限创建任务。

安装hexo-cli

D:>npm install hexo-cli -g

added 50 packages, and changed 3 packages in 5m

14 packages are looking for funding
run npm fund for details

hexo-cli 安装成功后显示如上信息。

退回到用户权限下，进入用户目录 c:\users\lenovo

执行hexo init mylog

失败，

进入myblog目录，

cd myblog

执行npm install

必须在myblog目录下，执行hexo g

形成静态网页。
